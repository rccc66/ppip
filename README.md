# 需要的变量

<br>ABUSEIPDB_API_KEY---abuseipdb.com查询IP网站的的api-key
<br>CCLOUDFLARE_API_TOKEN---Cloudflare的api-token
<br>CLOUDFLARE_ZONE_ID---Cloudflare的ZONE_ID(区域ID)
<br>CCLOUDFLARE_DNS_NAME", "us"---Cloudflare托管的域名DNS前缀
<br>CLOUDFLARE_DOMAIN---Cloudflare托管的域名
<br>FFOFA_COOKIE---Fofa的cookie
<br>19行代码中的FOFA_QUERY = 'server=="cloudflare" && header="Forbidden" && asn=="31898" && country=="US"'可以自己修改 搜索的关键词
